# Business Card Studio 개선점 정리

## 이번 수정에 포함한 항목

### 1. 왼쪽 설정 패널 재구성
- 상단에 현재 상태를 바로 읽을 수 있는 요약 카드 추가
- 섹션 바로가기 네비게이션 추가
- 기본 정보를 라벨 기반 그리드로 재배치해서 입력 흐름 정리
- 저장 영역을 별도 카드로 묶어 CTA와 상태 메시지 가독성 개선

### 2. 푸터 재정비
- 단순 텍스트 한 줄 구조를 브랜드 소개 + 핵심 기능 + 문의 정보 구조로 변경
- 브랜드 인지와 연락 경로가 한눈에 들어오도록 재배치

### 3. 저장 품질 및 미리보기 일치도 개선
- 저장 예상 해상도를 UI에 노출
- 카드 계산 스타일을 복제해서 html2canvas 렌더 정확도 보강
- 화면 크기에 따라 들쭉날쭉하던 저장 해상도를 일정한 고해상도 기준으로 보정
- 미리보기 프레임과 실제 카드 레이어를 분리해서 저장 결과와 시각 차이 축소

## 다음 우선순위로 추천하는 개선

### 1. 인쇄 대응 출력
- PNG 외에 PDF 출력 추가
- 재단선, 안전영역, bleed 가이드 추가
- 인쇄용 해상도 프리셋 제공

### 2. 이미지 편집 UX
- 업로드 이미지 크롭/확대/축소 UI 추가
- 배경 이미지 위치를 슬라이더 대신 캔버스 핸들로 조정
- 이미지가 카드 밖으로 얼마나 잘리는지 안전영역 시각화

### 3. 작업 히스토리
- 실행 취소 / 다시 실행 추가
- 최근 저장 상태 복원 강화

### 4. 템플릿 운영성
- 템플릿 즐겨찾기
- 템플릿 썸네일 품질 개선
- 앞면뿐 아니라 앞/뒤 세트 비교 지원

### 5. 저장 방식 고도화
- 앞면/뒷면 동시 저장
- 파일명 규칙 커스터마이징
- ZIP 묶음 다운로드

### 6. 데이터 입력 품질
- 전화번호/이메일 포맷 보정
- 필수값 누락 경고
- 너무 긴 텍스트에 대한 자동 줄바꿈 또는 축소 정책

### 7. 성능과 안정성
- 큰 이미지 업로드 시 로컬스토리지 한계 대응
- IndexedDB 기반 에셋 저장으로 전환 검토
- 느린 기기에서 비교 모드 렌더 최적화

### 8. 접근성과 디테일
- 키보드만으로도 조작 가능한 포커스 흐름 정리
- 버튼/상태 메시지 대비 강화
- 모바일에서 손가락 조작 영역 확대
